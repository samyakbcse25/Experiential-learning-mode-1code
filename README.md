# HTML Basics — Live Code Playground

I built this as part of my Experiential Learning project at
SB Jain Institute of Technology, Nagpur.

## What I built
An interactive HTML tutorial page where anyone can type HTML
code and see the output instantly — no external tools needed.

## How it works
- Left side has explanation of basic HTML tags
- Right side has a live editor — type and click Run
- Output appears inside the same page using an iframe

## Tags I used
DOCTYPE, html, head, body, h1-h6, p, b, i, a, img,
br, table, iframe, textarea
