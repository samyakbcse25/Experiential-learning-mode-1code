Document Structure

<!DOCTYPE html> — declares HTML5
<html> — root element, wraps all
<head> — metadata, title (invisible)
<body> — everything visible on page

Headings & Text

<h1> to <h6> — headings (h1 biggest)
<p> — paragraph of text
<b> — bold text
<i> — italic text
<br> — line break (no closing tag)

Links

<a href="url">text</a>
href = the URL to go to.
Opens on same page by default.

Images

<img src="file" alt="desc">
src = path to image file.
alt = text if image fails to load.
Self-closing — no </img> needed.
